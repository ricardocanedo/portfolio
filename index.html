Trabalhos

Este projeto foi feito para o curso de Projeto Integrador 3 da faculdade UNIVESP (2025) desenvolvido em Python com o framework Django e TypeScript com React. O objetivo era construir um site com sistema de blog integrado para veicular os artigos de um escritório de advocacia. Neste projeto pude aplicar as ferramentas mais modernas na construção de soluções para sistemas web e mobile. Participando de uma equipe fantástica ao lado de profissionais muito capacitados atuei em desafios interessantíssimos. Utilizando principalmente Laravel, Angular e Bootstrap construí módulos e funcionalidades para o sistema com mais de 150 mil usuários. Aqui pude fazer a modernização de endpoints legados que salvavam arquivos blob em mais de 84 tabelas do banco de dados. Utilizando o patter de Observables do Laravel desenvolvi uma solução que interceptava as chamadas dos models para estes arquivos, salvando na AWS S3 quando enviados ao servidor e injetando na requisição quando recuperados nas consultas. Construi um módulo de merendas que monta os cardápios dos municípios com base nos valores da tabela TACO e mostra toda a composição nutricional semanal total, com o carregamento dos cardápios personalizados e dinâmicos a depender da turma, escola ou registro acadêmico de cada aluno. Trabalhei em grande parte do módulo de registro de frequência dos funcionários e geração de relatórios. Criei diversas classes para uso geral do sistema, buscando o reaproveitamento de código. Também pude aplicar Angular junto com Ionic e Tailwind para desenvolver as funcionalidades da aplicação mobile do projeto. Meus maiores desafios neste projeto foram relacionados à personalização do sistema para diferentes municípios de modo que não afetasse os outros que utilizam as mesmas funcionalidades. Com as técnicas aprendidas durante este projeto, superar estes desafios foi enriquecedor.

Neste projeto eu tive o grande prazer de atuar com profissionais muito competentes e talentosas que me deram todo o suporte necessário para cumprir a função de programador web. Com o foco principal no backend, realizei a manutenção de CMS (Content Management System) e sistemas de ecommerces. Além da criação de novas funcionalidades para os sistemas da empresa, upgrade de versão do PHP de todos os sites, melhorias em sistemas de pagamentos e frete. E pude também trabalhar as funcionalidades de carregamento dinâmico dos conteúdos e imagens visando melhorar o SEO e ranqueamento das páginas. Atuei em funções para tratamento e redimensionamento das imagens de upload do site em diferentes dimensões e extensões, para carregamento dinâmico de imagens mais leves, melhorando o ranqueamento nos sistemas de busca. O maior desafio deste projeto foi trabalhar com o PHP puro em sistemas proprietários da empresa, mas que ao mesmo tempo me permitiu ampliar a visão de uso desta ferramenta tão versátil, que é o PHP.
